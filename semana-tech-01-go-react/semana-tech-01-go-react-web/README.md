# Iniciando a aplicação

```bs
npm create vite@latest
```

Selecione:

- web
- REACT
- Typescript

Entra na pasta e roda no VSCode

```bs
cd web
code .
```

Instala as dependências

```bs
npm i
```

```bs
npm i tailwindcss postcss autoprefixer -D
npx tailwindcss init -p
```

## Para iniciar a aplicação use

```bs
npm run dev
```
